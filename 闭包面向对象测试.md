小知识科普

**`&& 逻辑与 || 逻辑或`**

> 1、在条件判断中
> 
> + &&：所有条件都为真，整体才为真
> + ||：只要有一个条件为真，整体就为真
> 
> 2、在赋值操作中
> 
> + ||：A||B 首先看A的真假，A为真返回的是A的值，A为假返回B的值（不管B是啥）
> + &&：A&&B 首先看A的真假，A为假返回A的值，A为真返回B的值
> 
> **逻辑与的优先权高于逻辑或**
> 
> 项目中，常使用逻辑或实现默认值的操作

```javascript
function(num, callBack){
	callBack || callBack();
}
```
